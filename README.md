# Vue Commerce
A simple e-commerce app built with Laravel and Vue.js

## Getting Started
For Front-End
- run `cd vue && npm install` on your terminal
- run `npm run serve`
- open `http://localhost:3000`

For Back-End
- run `cd laravel && composer install`
- run `php artisan migrate && php artisan db:seed`
- run `php artisan serve`
- open `http://localhost:8000`
